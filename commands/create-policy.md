---
description: Start creating a new policy as Decision Manager
---

The user is a Decision Manager starting or continuing work on a policy. Load the policy-framework skill and go directly to the Decision Manager Guide — do not ask them to identify their role first, as they have already done so by running this command.

Begin by welcoming them briefly and asking which policy they're working on (or if they've mentioned it already, acknowledge it). Then move straight into Stage 1 of the Decision Manager flow: confirm they have the policy template open from Cognidox, and start helping them define the core content, scope, and decision points.

Work through the stages one at a time. Do not present the full framework upfront — guide them step by step, asking questions and waiting for responses before moving on.
